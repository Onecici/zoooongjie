// 滚动条整体部分
::-webkit-scrollbar {
    width: 6px; //对垂直方向滚动条
    height: 6px; //对水平方向滚动条
}

//滚动的滑块
::-webkit-scrollbar-thumb {
    border-radius: 3px;
    background-color: #ccc //滚动条的颜色
}

//内层滚动槽
::-webkit-scrollbar-track-piece {
    background-color: transparent;
}





注意！！！

一旦修改了::-webkit-scrollbar该属性就必须设置::-webkit-scrollbar-thumb和::-webkit-scrollbar-track-piece这些属性
